# Signal Processing MicroSims

Interactive Micro Simulations to help students learn signal processing fundamentals.

    ![Basic FFT](./fft-basic/fft-basic.png)

    ![Bloom's Taxonomy](./blooms-taxonomy/blooms-taxonomy.png)

- **[Book Levels](./book-levels/index.md)**


    ![Book Levels](./book-levels/book-levels.png)

    ![FFT Butterfly](./fft-butterfly/fft-butterfly.png)

    ![FFT From Microphone](./fft-mic/fft-mic.png)

- **[FFT Live Audio](./fft-live-audio/index.md)**


    ![FFT Live Audio](./fft-live-audio/fft-live-audio.png)

- **[FFT Size](./fft-size/index.md)**


    ![FFT Size](./fft-size/fft-size.png)

    ![FFT Sound File](./fft-sound-file/fft-sound-file.png)
